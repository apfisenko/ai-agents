## 🚀 Как запускать  

> **⚠️ Этот раздел будет дополняться.**  

```bash
# 1. Клонировать репозиторий
git clone https://github.com/apfisenko/ai-agents.git
cd ai-agents

# 2. Создать и активировать виртуальное окружение (рекомендовано)
python -m venv .venv
source .venv/bin/activate   # .venv\Scripts\activate on Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить пример (пример для М01)
python src/common/example_llm.py
```

*Подробные инструкции по запуску конкретных модулей появятся в соответствующих подпапках `assignments/` и `notebooks/`.*

### Jupyter: ядро для демо‑тетрадок

В репозитории используется отдельное виртуальное окружение **`.venv_notebook_demo`** (**Python 3.11**) и зарегистрированное ядро Jupyter / Cursor / VS Code. Интерпретатор 3.11 ставится [с python.org](https://www.python.org/downloads/) (x64) или `winget` — проверка: `py -3.11 --version`.

| | |
|---|---|
| **Имя ядра** | `ai-agents-notebook-demo` |
| **Отображаемое имя** | `Python 3.11 (ai-agents .venv_notebook_demo)` |
| **Интерпретатор** | `.venv_notebook_demo\Scripts\python.exe` (Windows) |

- Первичная настройка (создать venv при необходимости, установить `ipykernel`, зарегистрировать ядро): в корне репозитория выполнить **`register_jupyter_kernel.ps1`**. Скрипт переписывает путь к `python.exe` в спецификации ядра на **короткий формат 8.3** (через `…\03CB~1\…` и т.п.): иначе Cursor/VS Code иногда **падает при выборе ядра**, если путь к проекту содержит **не-ASCII** (например, кириллицу в `C:\FISENKO\ИИ\...`).
- Библиотеки для конкретной тетрадки при необходимости ставьте **внутри тетрадки** командой `%pip install …` — пакеты попадут в `.venv_notebook_demo`.
- Каталог **`.venv_notebook_demo/`** в `.gitinore` и в git не коммитится.
